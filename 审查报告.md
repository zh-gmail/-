## 剩余已知问题

### CRITICAL — 建议关注
1. **npm 依赖漏洞** — `npm audit` 报告 5 high / 1 moderate。esbuild 漏洞需等 Vite 升级到 esbuild 0.28+，当前 vite@6.4.3 依赖 esbuild@0.25.x。均为**构建时漏洞**，正常运行不可利用

### MAJOR — 建议修复
2. **Context 过大导致全树重渲染** — 所有状态在同一个 Context 中，任何字段变化触发所有 5 个 Tab 重渲染。建议拆分为多个 Context 或使用 `useMemo` 隔离
3. **Tab 组件每次切换卸载重建** — 切回 LiveCamera 需重新初始化 AR 引擎（3-15 秒）。可用 CSS `display:none` 替代条件渲染

### MINOR
4. API polling 未使用 AbortController — 组件卸载后请求仍继续（但 React 已处理，无实际影响）
5. 无测试覆盖 / 无 CI/CD
